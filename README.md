# Usage

simply:

        % git clone https://github.com/uwabami/zsh-config <path>
        % ln -s <path>/zshenv ~/.zshenv
        % ln -s <path>/zsh.d ~/.zsh.d
        % cd ~/.zsh.d
        % git submodule init
        % git submodule update
        % make
